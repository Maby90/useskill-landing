# Skill di useskill.it — Guida all'installazione

Hai appena scaricato una o più Skill. Questa guida ti spiega come installarle e iniziare a usarle in meno di un minuto.

---

## Cosa c'è nel pacchetto

Ogni Skill è un file `.skill` che contiene le istruzioni per l'AI. In aggiunta trovi:

Il file **PROFILE.md** è il tuo profilo professionale. Lo compili una volta con le tue informazioni (settore, tono di voce, target, piattaforme) e tutte le Skill lo leggono per personalizzare l'output su di te. Non è obbligatorio ma fa una differenza enorme.

Se hai la Skill **Brand Voice Extractor** (gratuita), usala come primo step: analizza i tuoi contenuti esistenti e compila il profilo per te.

---

## Installazione rapida

### Su Claude (claude.ai)

1. Vai su claude.ai e apri la sezione **Progetti**
2. Crea un nuovo progetto (es. "Le mie Skill")
3. In **Conoscenza del progetto**, carica tutti i file `.skill` che hai acquistato
4. Carica anche il `PROFILE.md` (compilato)
5. Apri una conversazione nel progetto e inizia a lavorare

Tutte le Skill che hai caricato funzionano insieme nello stesso progetto. Nessun conflitto: ognuna si attiva solo quando serve.

### Su Google Antigravity

1. Apri una nuova sessione
2. Carica i file `.skill` e il `PROFILE.md` come contesto della sessione
3. Scrivi il tuo prompt — l'agente segue le istruzioni

### Su Manus o altri agenti AI

> **Nota:** Manus accetta solo file con estensione `.skill`. Prima di caricare, rinomina il file `SKILL.md` in `SKILL.skill` (su Mac: tasto destro → Rinomina; su Windows: F2). Il contenuto non cambia, solo l'estensione.

1. Aggiungi il contenuto dei file `.skill` come istruzioni di sistema o contesto
2. Aggiungi il `PROFILE.md` per la personalizzazione
3. Lavora normalmente — l'agente applica le Skill

## Ordine consigliato per iniziare

Se hai il pacchetto completo, l'ordine ideale è:

**Primo:** Brand Voice Extractor. Incolla 5-10 tuoi contenuti (post, email, newsletter). La Skill analizza il tuo tono e ti chiede se vuoi compilare il PROFILE.md con i risultati. Dì di sì.

**Secondo:** Compila le parti del PROFILE.md che la Brand Voice Extractor non ha coperto (settore, target, piattaforme, obiettivi). Ci vogliono 5 minuti.

**Terzo:** Usa qualsiasi altra Skill. L'output sarà già calibrato su di te.

## Domande frequenti

**Le Skill funzionano solo con Claude?**
No. Funzionano con qualsiasi AI che accetta istruzioni di contesto: Claude, Google Antigravity, Manus, e altri. Anche sui piani gratuiti.

**Devo essere tecnico per usarle?**
No. Se sai incollare un file in una chat, sai usare una Skill. Nessun codice, nessuna configurazione.

**Posso usare più Skill insieme?**
Sì. Caricale tutte nello stesso progetto (su Claude) o nella stessa sessione. Ognuna si attiva solo quando serve.

**Il PROFILE.md è obbligatorio?**
No, ma fa la differenza tra un output generico e uno calibrato su di te. Vale i 5 minuti che servono per compilarlo.

**Posso modificare le Skill?**
I file .skill sono testo — puoi aprirli e leggerli. Ma le modifiche sono a tuo rischio: la struttura è ottimizzata per funzionare così com'è.
